Public Docker Stack

This site is served from a containerized Nginx instance running behind Traefik with automatic HTTPS. It is part of a publicly documented, production-style Docker stack used to demonstrate modern infrastructure patterns.

The stack focuses on security, observability, and automation, using infrastructure-as-code and environment-based configuration.

Included Services

Docker Compose
Traefik
Cloudflare DNS
Prometheus
Grafana
Loki
Pi-hole
Nginx

Key Characteristics

  • No secrets committed to source control
  • Environment-variable driven configuration
  • Automatic TLS via DNS-01 challenge
  • Centralized logging and metrics
  • Reverse proxy with secure defaults
